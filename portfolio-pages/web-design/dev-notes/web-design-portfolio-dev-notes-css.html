common breakpoints

media query:

@media screen and (min-width:768px) {}

  • 480px
  • 768px
  • 1024px
  • 1280px

em

1em = element level font size

equal to the font-size of the current element or the default base font-size set by the browser if none is given

Example:

  • font-size: 16px;
  • 1em = 16px
  • 2em = 32px

rem

1rem = root level font size

rem stands for root em

equal to the font-size of the root element, <html>, whereas em checks for the parent elements to font-size.

Example:

  • font-size: 16px;
  • 1rem = 16px
  • 2rem = 32px
